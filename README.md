## ‚ú® Key Features

- [x] **Bidirectional Infinite Loop:** Provides seamless, continuous scrolling in both directions.
- [x] **Custom Paging:** Custom snap-to-center behavior without relying on standard `isPagingEnabled`.
- [x] **Custom Layout:** Fully customizable item size and spacing.

## Requirements
- UIKit
- iOS 13+

- Select "Up to Next Major" with "1.0.1"
